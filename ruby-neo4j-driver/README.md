## Ruby Driver

A simple Ruby program that test drives the Neo4j Ruby Driver.

## Driver

[neo4j-core][ruby-driver] (**Community Driver**) | [Docs][ruby-driver-docs]

### Setup

```bash
bundle install
bundle exec ruby test_social.rb
```
